## About

**Power BI Website DAX Guide** is a web-based educational platform aimed at helping users master advanced techniques in Power BI, with a special focus on DAX. This guide provides:

- Step-by-step tutorials for complex DAX syntax
- Strategic guidance on using measures versus calculated columns
- Advanced data modeling techniques for optimal report-building

---

## Features

- 🧑‍💻 **Comprehensive DAX Tutorials**: Move beyond the basics with in-depth guides.
- 🔢 **Measures vs. Calculated Columns**: Learn best-use scenarios for each.
- 📊 **Data Modeling Best Practices**: Understand relationships, star/snowflake schemas, and performance tuning.
- 🔍 **Searchable Knowledge Base**: Quickly find solutions, patterns, and reference material.
- 🚀 **Modern Responsive UI**: Built with TypeScript, the interface is fast and user-friendly.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) >= 14.x
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/eaearly/Power-BI-Website-DAX-Guide.git
   cd Power-BI-Website-DAX-Guide
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   or
   ```bash
   yarn dev
   ```

   Visit [http://localhost:3000](http://localhost:3000) to view the website locally.
